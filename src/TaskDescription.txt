Generics и списки.

Описание задания
#1 Создать класс для математических вычислений: Содержит 4 статических метода на сложение, вычитание, умножение,
деление. Обработать ошибку деления на ноль. Методы должны быть Generic и принимать данные в разных форматах
(строка, число).

#2 Создать дженерик класс User, у которого поля id, card_number могут иметь типы String или int

#3 Реализовать метод, который принимает на вход список элементов любого типа данных, создает список из 100 элементов
заполненный элементами из переданного списка в случайном порядке (Загуглить генерация случайных чисел Java)

#4** Создать две фабрики для юнитов разных типов. Создать метод, который будет создавать армию (список) юнитов
созданных одной фабрикой (фабрика должна генерировать параметры юнитов со случайным разбросом) и реализовать метод бой,
который в случайном порядке будет сталкивать юнитов из двух армию (списков). Печатать подробный лог боя
